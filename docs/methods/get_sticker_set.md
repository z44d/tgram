#🔧 get_sticker_set

**Use this method to get a sticker set. On success, a StickerSet object is returned.**

##⚙️ Parameters

- **`name`** (**`str`** ): **Sticker set name**

##📲 Returns

#### [StickerSet](../types/StickerSet.md)

##📀 Examples

-🪫 **Required Parameters**

```python
await bot.get_sticker_set(
    name=your_name_here
)
```
