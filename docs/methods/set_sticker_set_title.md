#🔧 set_sticker_set_title

**Use this method to set the title of a created sticker set.**

##⚙️ Parameters

- **`name`** (**`str`** ): **Sticker set name**
- **`title`** (**`str`** ): **New sticker set title**

##📲 Returns

#### `bool`

##📀 Examples

-🪫 **Required Parameters**

```python
await bot.set_sticker_set_title(
    name=your_name_here,
    title=your_title_here
)
```
