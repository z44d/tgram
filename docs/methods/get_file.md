#🔧 get_file

**Use this method to get basic info about a file and prepare it for downloading.**

##⚙️ Parameters

- **`file_id`** (**`str`** ): **File identifier**

##📲 Returns

#### [File](../types/File.md)

##📀 Examples

-🪫 **Required Parameters**

```python
await bot.get_file(
    file_id=your_file_id_here
)
```
